╔══════════════════════════════════════════════════════════════════════════════╗
║ 🚨 SOLUCIONAR PROBLEMAS                                                     ║
╚══════════════════════════════════════════════════════════════════════════════╝

PROBLEMA: "El puerto 3000 ya está en uso"
SOLUCIÓN: 
  • Cambiar puerto en backend/server.js (línea 7)
  • O cerrar el programa usando el puerto

PROBLEMA: "Cannot find module 'express' o similar"
SOLUCIÓN:
  • Ejecutar: cd backend && npm install
  • Verificar que NodeModules exista

PROBLEMA: "La página se ve sin estilos"
SOLUCIÓN:
  • Recarga con Ctrl+F5 (limpiar caché)
  • Verify servidor está corriendo

PROBLEMA: "No puedo iniciar sesión"
SOLUCIÓN:
  • Verifica usuario y contraseña exactamente
  • Revisa archivo backend/database/users.json
  • Reinicia el servidor

PROBLEMA: "Las noticias no aparecen en el sitio público"
SOLUCIÓN:
  • Verifica que guardaste correctamente
  • Recarga la página
  • Revisa que estado sea "activo"
